pygments_highlighter Module¶

pyface.ui.qt4.code_editor.pygments_highlighter.get_tokens_unprocessed(self, text, stack=('root', ))¶

Split text into (tokentype, text) pairs.

Monkeypatched to store the final stack on the object itself.

The text parameter that gets passed is only the current line, so to highlight things like multiline strings correctly, we need to retrieve the state from the previous line (this is done in PygmentsHighlighter, below), and use it to continue processing the current line.
